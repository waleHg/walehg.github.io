Overview

I was contacted to work on the design of a Mobile Application for Klassconnect, a learning management system. The Klassconnect team wanted to build a mobile app with same features available on the existing web application of the learning management system and I was tasked with redesigining and replicating the existing features on a mobile app ensuring the experience is consistent on both the mobile & web platforms.

Details

ROLE IN PROJECT

User Experience Design

Challenges

1. Since a web platfrom already existed, the design for the mobile app needed to match that of the web platform.

2. Two mobile applications were planned to be designed in a short time frame, one for students and another for lecturers with the same features on the web platform.

Research

I conducted interviews with stakeholders to understand project needs and also interviewed some students who use similar learning tools to understand their pain points while using such tools. I also explored other similar applications to understand how they work and to ascertain how best to design features on the mobile app. This was important to have a grasp of the current experience on the platform to provide direction for the user flow and help guide the design and assess best ways to visually represent features on the application.

User Flow

Colours

I adapted the primary colour from the existing web platform in the design for the Student App to ensure a consistent experience across both platforms. The primary colour (blue) is a good choice for the design because it symbolizes wisdom, intelligence, and truth. Additionally, I used a dark grey colour matching the primary colour for body text and and used white backgrounds to allow elements such as links and heading text stand out.

Typography

Simplicity was key to the design so I decided to use only one typeface. Roboto was the selected typeface because of its legibility and because it is the default android typeface. Also, because the typeface on the existing platform is Raleway (a sans-serif font) which is a similar font from the sans-serif family - Roboto was chosen to ensure consistenty across mobile and web.

Conclusion

Simplicity was key to the design of the app because it had quite a number of features, so based on my research I ensured that the most used features were placed in a bottom navigation menu and so users can easily navigate through the most used features.

Other features were placed in an hamburger menu and each had a descriptive icon so it's easy for a user to find a feature by merely looking at the icons.
