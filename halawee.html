Halawee

KlasConnect is a revolutionary learning management system that aims to make interaction between lecturers and students easier. I worked on designing the User Interface for the Mobile App.

Overview

I designed the user interface for the admin dashboard with features such as viewing latest transactions, total value of transactions, customers, all transactions and their details, admin profile and settings. I also converted the user interface designs into HTML/CSS code using Bootstrap Material template.

The Challenge

Two mobile applications were planned to be designed, one for Students and another for Lecturers. Both applications were to have the same features which on the existing web platform.

Research

I conducted interviews with stakeholders to understand project needs and also interviewed some students who use similar learning tools to understand their pain points while using such tools. I also explored other similar applications to understand how they work and to ascertain how best to design features on the mobile app. This was important to have a grasp of the current experience on the platform to provide direction for the user flow and help guide the design and assess best ways to visually represent features on the application.

Colours

On the existing platform, blue is used as the primary colour so I adapted this in the design for the Student App to ensure a consistent experience across both platforms. The color blue is a good choice for the design because it symbolizes wisdom, intelligence, and truth. Blue is also considered beneficial to the mind and produces a calming effect. Additionally, I used a dark grey colour matching the primary colour for body text and and used white backgrounds to allow elements such as links and heading text stand out.

Typography

Simplicity was key to the design so I decided to use only one typeface. Roboto was the selected typeface because of its legibility and because it is the default android typeface. Also, because the typeface on the existing platform is Raleway a sans-serif font a similar font from the sans-serif family - Roboto which was chosen to ensure consistenty across mobile and web.

Conclusion

Simplicity was key to the design of the app because it had quite a number of features, so based on my research I ensured that the most used features were placed in a bottom navigation menu and so as to prevent information overload

Other features were placed in an hamburger menu and each had a descriptive icon so it's easy for a user to find a feature by merely looking at the icons.

The interfaces for the student and lecturer apps were similar but I made use of different colours for both apps to make them distinct
