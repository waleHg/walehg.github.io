Owner Direct

Owner Direct is a property lising platform that helps Landlords find tenants without the hassle of an agent and tenants find their dream house without any agent acting as a middleman.

Introduction

Aiming to be a platform that not only lists properties but make it easy for people to also list properties, having obvious call to actions for property owners and people searching for property, and also communicating the value of the platform is important to the overall user experience.

Problem

Most property listing websites available in Nigeria just load many lisings on their platform and do not properly segment available listings, this leads to information overload and makes it difficult for users to find properties. The goal was to

1. Design a search experience that guides the user through searching through a location, property type, and selecting a budget.
2. Properly segment and display listings to minimise clutter and make it easy for users to find properties.

Designed Solution

1. Searching for a property Search flow Type a location --> Select propert Type --> Select a Budget (Property price range) **Top search locations can be seen by users just below the search fields-This allows for a user to skip the step of entering a location if it is a top search location **Users can also quickly toggle between searching for a property "For Rent" or 'For Sale"

2. Search Results -The first section on the search result page has a count of the number of properties found in a location -Search filters are also available and stay stickly on scroll so a user can easily use filters even when they have scrolled down on the page -Lisings are arranged in cards with the Price visibly larger and obvious, the card bears an image of the property, the type of property, the location of the property as well as ammenities available.

3.Viewing a property -Clicking on a card loades the page that shows full details of the property, a call to action button to contact who listed the property, a grid that shows images of the property, a button to add property to a user's bookmarks on the platform and the full description of the property.
